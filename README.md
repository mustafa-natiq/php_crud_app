# php_crud_app

a simple CRUD app written in vanilla php

# Start server

Run `php -S 127.0.0.1:5000 -t public`

```
php_crud_app

├─ public
│  ├─ form.html
│  ├─ index.php
│  └─ processor.php
├─ README.md
└─ setup.sql

```
